# Overview

This document defines all reusable UI components used throughout the MY ACHIEVERS TRUST website.

Every component must be:

- Reusable
- Responsive
- Accessible
- Consistent
- Easy to maintain
- Animation-ready

Never duplicate components if they can be reused.
